# Changelog

All notable changes to the coordination-toolkit (`cotk`) plugin will be documented in this file.

## [1.0.2] - 2026-06-14 — first open-source release

Multi-session peer coordination: peer discovery, file claims, conflict detection, task distribution, and an opt-in real-time messaging bridge. 4 skills, 4 commands.

### Highlights

- Filesystem-based peer registry so multiple Claude Code sessions can coordinate without silently editing the same files.
- MIT licensed.

_First public release at 1.0.2; earlier version history was internal and has been omitted._
